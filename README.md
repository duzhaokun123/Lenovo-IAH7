# Lenovo IAH7

https://github.com/johnfanv2/LenovoLegionLinux 的替代

功能:

- 风扇转数

你也许应该使用这个模块而不是 LLL

因为其实 LLL 在 `Lenovo Legion Y9000X (2022) IAH7/82TF` BIOS: `JYCN58WW` EC: `JYEC55WW` 上根本不能正常工作
用这个至少你拿到了正确的风扇转速

根据 [Lenovo 境外官网信息](https://pcsupport.lenovo.com/us/en/products/laptops-and-netbooks/legion-series/legion-s7-16iah7/82tf) `Legion S7 16IAH7` 应该为同一机型

在 Linux 上其实也可能更新 BIOS
参考 https://wiki.archlinux.org/title/Flashing_BIOS_from_Linux#Lenovo
