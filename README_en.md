# Lenovo IAH7

An alternative to https://github.com/johnfanv2/LenovoLegionLinux

Features:

- Fan RPM

You might want to use this module instead of LLL.

As LLL simply does not work on `Lenovo Legion Y9000X (2022) IAH7/82TF` which BIOS: `JYCN58WW` EC: `JYEC55WW`
With this, you get correct fan speed at lset.

According to the [Lenovo international support page](https://pcsupport.lenovo.com/us/en/products/laptops-and-netbooks/legion-series/legion-s7-16iah7/82tf), `Legion S7 16IAH7` should be the same hardware variant.

It is also possible to update the BIOS from Linux.
See: https://wiki.archlinux.org/title/Flashing_BIOS_from_Linux#Lenovo
