# Lenovo IAH7

Alternative to https://github.com/johnfanv2/LenovoLegionLinux

Features:

- Fan RPM

You should probably use this module instead of LLL

As LLL simply does not work on `Lenovo Legion Y9000X (2022) 82TF/IAH7 JYCN56WW`
With this, you get correct fan speed at lset.
